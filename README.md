> 💡 **Примечание:** Все задания из Python FastAPI адаптированы под Node.js + Express.js

---

## 🎱 Установка

```bash
npm install
```

## 🦄 Запуск

### Обычный режим:
```bash
npm start
```

### Режим разработки (с автоперезагрузкой):
```bash
npm run dev
```

Сервер запустится на `http://localhost:8080`
